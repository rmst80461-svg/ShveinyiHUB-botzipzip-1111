# Развёртывание на Bothost.ru

## Важно
На Bothost работает **только Telegram бот**. Веб-админка недоступна — используйте админ-функции в боте (`/admin`) или держите веб-панель на Replit.

## Требования
- Аккаунт на [bothost.ru](https://bothost.ru)
- GitHub/GitLab репозиторий с кодом бота

## Шаги развёртывания

### 1. Подключите репозиторий
1. Зарегистрируйтесь на bothost.ru
2. Создайте новый проект
3. Подключите ваш GitHub/GitLab репозиторий

### 2. Настройте переменные окружения
В панели управления Bothost добавьте следующие переменные:

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен Telegram бота от @BotFather |
| `GIGACHAT_CREDENTIALS` | API ключ GigaChat (Sber AI) |
| `DATABASE_URL` | URL базы данных PostgreSQL |
| `ADMIN_ID` | Telegram user ID администратора |

### 3. База данных
- Используйте внешний PostgreSQL (Neon, Supabase, ElephantSQL)
- При отсутствии DATABASE_URL бот использует SQLite (данные теряются при перезапуске)

### 4. Деплой
После настройки переменных окружения сделайте push в main ветку — Bothost автоматически соберёт и запустит проект.

## Управление через бота
Вместо веб-админки используйте Telegram:
- `/admin` — войти в админ-панель
- Просмотр заказов, статистика, рассылка — всё доступно в боте

## Структура файлов
```
├── Procfile              # worker: python main.py
├── requirements.txt      # Python зависимости
├── main.py               # Telegram бот
└── handlers/             # Обработчики бота
```

## Минимальные требования
- RAM: 256 MB (бесплатный тариф подойдёт)
- Рекомендуется: Basic (99₽/мес) для env переменных и бэкапов
