# Развёртывание на Bothost.ru

## Требования
- Аккаунт на [bothost.ru](https://bothost.ru)
- GitHub/GitLab репозиторий с кодом бота

## Шаги развёртывания

### 1. Подключите репозиторий
1. Зарегистрируйтесь на bothost.ru
2. Создайте новый проект
3. Подключите ваш GitHub/GitLab репозиторий

### 2. Настройте переменные окружения
В панели управления Bothost добавьте следующие переменные:

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен Telegram бота от @BotFather |
| `GIGACHAT_CREDENTIALS` | API ключ GigaChat (Sber AI) |
| `DATABASE_URL` | URL базы данных PostgreSQL |
| `ADMIN_ID` | Telegram ID администратора |
| `ADMIN_PASSWORD` | Пароль для веб-админки |
| `FLASK_SECRET_KEY` | Секретный ключ для Flask сессий |

### 3. База данных
- На тарифе Basic и выше доступны managed базы данных
- Либо используйте внешний PostgreSQL (например, Neon, Supabase)
- При отсутствии DATABASE_URL бот использует SQLite (данные теряются при перезапуске на бесплатном тарифе)

### 4. Порты
- Веб-админка запускается на порту 5000
- Telegram бот работает через polling (не требует открытого порта)

### 5. Деплой
После настройки переменных окружения сделайте push в main ветку — Bothost автоматически соберёт и запустит проект.

## Структура файлов
```
├── Procfile              # Конфигурация запуска
├── requirements.txt      # Python зависимости
├── run_services.py       # Запуск бота + веб-админки
├── main.py               # Telegram бот
├── webapp/               # Flask админка
└── handlers/             # Обработчики бота
```

## Минимальные требования
- RAM: 256 MB (бесплатный тариф подойдёт для тестирования)
- Рекомендуется: Basic (99₽/мес) для постоянного хранения данных
